# ResQRail 🚆

ResQRail is an intelligent transit planning application designed to help Indian railway passengers find alternative travel options when their train tickets are waitlisted, cancelled, or unavailable.

The platform analyzes available transportation modes and recommends optimized routes based on cost, travel time, convenience, and reliability.

---

## Features

### Smart Alternative Route Planning

* Suggests alternative travel routes when train tickets are unavailable.
* Combines multiple transportation modes for seamless journeys.

### Multi-Modal Transportation

Supports:

* Indian Railways
* State Transport Buses
* Private Bus Operators
* Metro Networks
* Ride Sharing Services
* Flights (future integration)

### Intelligent Recommendations

* Travel time estimation
* Cost comparison
* Route optimization
* Transfer analysis

### User-Friendly Interface

* Clean and modern Android UI
* Fast route generation
* Mobile-first experience

### Emergency Travel Assistance

Helps passengers quickly identify alternate travel options during:

* Waitlisted bookings
* Train cancellations
* Missed connections
* Unexpected disruptions

## Installation

### Prerequisites

* Android Studio
* JDK 17+
* Android SDK

### Steps

1. Clone the repository

```bash
git clone https://github.com/your-username/ResQRail.git
```

2. Open the project in Android Studio.

3. Sync Gradle dependencies.

4. Build the project.

```bash
Build → Build APK(s)
```

5. Install the generated APK on your Android device.
